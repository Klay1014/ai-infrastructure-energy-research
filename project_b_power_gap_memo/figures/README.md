# Figures

These are editable figure drafts for the Project B memo. They are explanatory diagrams, not fake-precise forecasts.

## Figure List

1. `01_temperature_power_chain.mmd`
   - Explains how temperature becomes a power-system constraint.

2. `02_power_adequacy_layers.mmd`
   - Separates annual energy, peak capacity, night peak, clean power, and resilience gaps.

3. `03_public_signal_dashboard.mmd`
   - Shows the public signals to track quarterly.

4. `04_scenario_matrix.md`
   - Frames risk by demand growth and supply/grid delivery.

5. `05_risk_matrix.md`
   - Links temperature conditions to electricity-system signals.
